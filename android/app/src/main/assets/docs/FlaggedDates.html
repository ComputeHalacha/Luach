A Flagged Date is an Onah that probably needs to be observed with Harchaka restrictions (הרחקות).

There are a few different situations that cause Flagged dates.

  • The 30th day after the beginning of a period
  • The same day of the month as a period for the month following a period. This is known as the Yom Hachodesh
  • There are many that also observe the 31st day after the beginning of a period
  • If there is an active Kavuah, it will generate Flagged Dates.
    Which dates need to be observed, depend on the type of Kavuah.
